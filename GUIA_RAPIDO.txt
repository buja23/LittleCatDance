🎮 COMO JOGAR COM AMIGOS (3 PASSOS):

┌─────────────────────────────────────────────────────────┐
│  PASSO 1: CRIAR/ENTRAR NA SALA                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  VOCÊ (Host):                                           │
│  ✓ Digite seu nome                                      │
│  ✓ Clique "Criar Nova Sala"                            │
│  ✓ Veja o código (ex: ABC123)                          │
│  ✓ Envie o código para amigos                          │
│                                                          │
│  SEUS AMIGOS:                                           │
│  ✓ Digitam o nome                                       │
│  ✓ Digitam o código: ABC123                            │
│  ✓ Clicam "Entrar na Sala"                             │
│                                                          │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  PASSO 2: COMEÇAR A JOGAR                               │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  ✓ Todos na sala veem o botão "Começar a Jogar"        │
│  ✓ Qualquer um pode começar quando quiser              │
│  ✓ Pressione qualquer tecla ou clique no botão         │
│                                                          │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  PASSO 3: COMPETIR!                                     │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  🎵 Maestro (laranja) mostra o movimento                │
│  👤 Você (azul) copia o movimento                       │
│  ⌨️  Use WASD ou Setas (↑←↓→)                          │
│  ⏱️  Acerte no tempo certo (veja a barra)              │
│  🏆 Quem fizer mais pontos ganha!                       │
│                                                          │
└─────────────────────────────────────────────────────────┘


⌨️  CONTROLES:
═══════════════════════════════════════════════════════════
  W ou ↑  = Braços para CIMA
  S ou ↓  = Braços para BAIXO
  A ou ←  = Braços para os LADOS
  D ou →  = Braços CRUZADOS


📊 PONTUAÇÃO:
═══════════════════════════════════════════════════════════
  ⭐ EXCELENTE (centro verde)  = +50 pontos
  👍 BOM (zonas azuis)         = +20 pontos
  🔥 COMBO (sequência)         = Bônus extra!


❤️  VIDAS:
═══════════════════════════════════════════════════════════
  Você tem 3 vidas (❤️❤️❤️)
  
  Perde vida se:
  ❌ Errar o movimento
  ❌ Não apertar nada a tempo
  ❌ Apertar no tempo errado
  
  💀 Perdeu tudo? Jogue de novo na mesma sala!


🏆 RANKING AO VIVO:
═══════════════════════════════════════════════════════════
  Veja a pontuação de todos em tempo real:
  
  🥇 1º Lugar  = Medalha de ouro
  🥈 2º Lugar  = Medalha de prata
  🥉 3º Lugar  = Medalha de bronze
  ✅ Vivo      = Ainda jogando
  💀 Morto     = Pode jogar de novo


💡 DICAS IMPORTANTES:
═══════════════════════════════════════════════════════════
  ✓ Código da sala = 6 caracteres (letras e números)
  ✓ Compartilhe pelo WhatsApp, Discord, etc.
  ✓ Não feche a aba do navegador!
  ✓ Precisa de internet funcionando
  ✓ Sem limite de jogadores por sala
  ✓ Cada um joga no seu ritmo (vidas independentes)
  ✓ Use a barra de timing como guia visual


🔧 RESOLVENDO PROBLEMAS:
═══════════════════════════════════════════════════════════
  "Sala não encontrada"
  → Verifique o código (maiúsculas/minúsculas não importam)
  
  "Firebase não disponível"
  → Verifique sua internet e recarregue (F5)
  
  "A sala foi fechada"
  → Todos saíram. Crie uma nova sala!
